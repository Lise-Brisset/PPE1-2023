Mini Projet

Le mini projet s'effectue dans le cadre du cours Programmation et Projet Encadré (PPE1). Le code de ce cours est L7TI005-P. Ce projet est dans la continuité de la formation pluriTAL enseigné sur un ensemble de trois universités qui sont la Sorbonne Nouvelle, l'INALCO et Nanterre Université.

Présentation du mini projet :

L'objectif de ce projet est de faire aux étudiants la première partie du futur projet seul. Ceci afin que les professseurs, M.Dupont et M.Magistry s'assurent que chacun ait essayé et compris les débuts et bases du projet encadré.

Le projet encadré :

Le projet encadré se fait en groupe de trois étudiants. Chaque étudiant se devra de maitrisé au moins une langue différente de celle de ses camarades de groupe. Il etait bénéfique de former un groupe de trois personnes provenant de trois université différentes. Ceci afin que chacun puisse apporter des connaissances qui lui sont propre aux autres.

Dans le cadre du projet, chaque groupe choisi un terme qu'il serait interessant d'étudier dans les trois langues. Les utilisations de ce mot seront ensuite comparées par le biai de l'analyse de contexte dans des sites internets. Pour chaque langue, il est demandé d'analyser une cinquantaine de liens.

Comment est-il construit le mini projet ?

Le mini projet est un entrainement des actions que les étudiants feront dans le cadre du grand projet, ceci sur la base d'un fichier txt de cinq urls. Voici les étapes principales :

  • Parcourir les urls avec la commande curl dans bash ;
  • Retrouver les éléments intéressants tels que le code HTTP, l'encodage de la page ;
  • Placer ces informations dans un format tabulé ;
  • Transformer ces lignes tabulés en format tableau dans un fichier HTML ;
  • Le tout grâce à un script bash qui est lancé dans le terminal.
